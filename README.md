## Overview
FastAPI-based CRUD service with PostgreSQL.

## Features
- RESTful CRUD
- Validation with Pydantic
- Proper HTTP status codes
- PostgreSQL persistence

## Tech Stack
FastAPI, SQLAlchemy, PostgreSQL

## Endpoints
POST /items
GET /items
PUT /items/{id}
DELETE /items/{id}

## How to Run
1. Clone
2. Create virtualenv
3. Install requirements
4. Run uvicorn
